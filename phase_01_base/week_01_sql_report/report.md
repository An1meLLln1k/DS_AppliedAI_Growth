# Week 01 SQL Mini Report

## Dataset

The report is based on two tables:

- `users`: user profile data
- `orders`: user order data

## Goal

Summarize basic customer and order behavior using SQL.

## Key Metrics

- Users count: 6
- Orders count: 8
- Total order amount: 211700
- Average order amount: 26462.5

## Key Findings

1. The highest total order amount belongs to Dmitry: 100000.
2. The city with the highest total amount is Moscow: 190000.
3. The segment with the highest average order amount is premium.
4. The top expensive products are laptop, phone and monitor.

## SQL Topics Used

- SELECT
- JOIN
- GROUP BY
- CTE
- CASE WHEN
- Window functions
- Subqueries
- Aggregations

## Result

This mini-report summarizes customer value, order distribution by city and segment, and top orders using SQL queries from Week 01.
